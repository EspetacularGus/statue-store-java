O aplicativo .jar para execução está em 'out -> artifacts'

A maneira mais recomendável de visualizar o projeto e as classes é pela IntelliJ IDE,
a mesma pode ser baixada gratuitamente em :
https://www.jetbrains.com/idea/download/#section=windows

Para os anúncios funcionarem corretamente, esta pasta "Entrega 2019 (Statue Store)",
com a aplicação Java e Web nela, deve estar em 
C:\Users\User\Desktop\ 
para conseguir simular corretamente o caminho do que seria o servidor
